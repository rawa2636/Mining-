# 🪙 محلل الذهب الجيولوجي بالذكاء الاصطناعي (Gold Analyzer AI)

هذا التطبيق هو أداة متقدمة تعتمد على الذكاء الاصطناعي (Gemini 3.1 Pro) لتحليل العينات الجيولوجية والصخور لاكتشاف مؤشرات وجود الذهب، مع التركيز على أكاسيد الحديد وعروق المرو والتشققات.

التطبيق مبني باستخدام:
- **React & Vite** (واجهة المستخدم)
- **Tailwind CSS** (التصميم)
- **Firebase** (قاعدة البيانات وتسجيل الدخول)
- **Google Gemini API** (الذكاء الاصطناعي للتحليل)

---

## 🚀 دليل النشر المجاني على Vercel

يمكنك استضافة هذا التطبيق مجاناً بالكامل على منصة Vercel. اتبع الخطوات التالية بدقة:

### الخطوة 1: رفع الكود إلى GitHub
1. قم بإنشاء حساب في [GitHub](https://github.com/).
2. قم بإنشاء مستودع جديد (New Repository).
3. ارفع جميع ملفات هذا المشروع إلى المستودع الخاص بك.

### الخطوة 2: النشر على Vercel
1. اذهب إلى موقع [Vercel](https://vercel.com/) وقم بتسجيل الدخول باستخدام حساب GitHub الخاص بك.
2. من لوحة التحكم، اضغط على زر **Add New...** ثم اختر **Project**.
3. قم باستيراد (Import) المستودع الذي قمت بإنشائه في الخطوة السابقة.
4. **⚠️ خطوة هامة جداً (المتغيرات البيئية):**
   في صفحة إعدادات النشر (Configure Project)، افتح قسم **Environment Variables** وأضف المتغير التالي:
   - **Name:** `GEMINI_API_KEY`
   - **Value:** (ضع مفتاح API الخاص بك من Google AI Studio هنا)
5. اضغط على زر **Deploy**.
6. انتظر دقيقة واحدة، وسيعطيك Vercel رابطاً مجانياً لتطبيقك (مثال: `your-app.vercel.app`).

### الخطوة 3: إعداد Firebase (مهم جداً لعمل تسجيل الدخول)
بما أن التطبيق يستخدم تسجيل الدخول بحساب جوجل، يجب أن تخبر Firebase أن رابط Vercel الجديد هو رابط "آمن":
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/).
2. ادخل إلى مشروعك، ثم من القائمة الجانبية اختر **Authentication** (المصادقة).
3. اذهب إلى تبويب **Settings** (الإعدادات) ثم **Authorized domains** (النطاقات المصرح بها).
4. اضغط على **Add domain** وأضف رابط Vercel الجديد الخاص بك (بدون `https://` وبدون `/` في النهاية. مثال: `your-app.vercel.app`).

## 💰 كيفية إضافة الإعلانات (AdSense أو غيرها) وجني الأرباح

لقد تم تجهيز أماكن استراتيجية للإعلانات داخل التطبيق (Ad Placeholders) لضمان أعلى نسبة نقر (CTR) دون إزعاج المستخدم.

### الخطوات:
1. قم بالتسجيل في شبكة إعلانية مثل **Google AdSense** أو **PropellerAds** أو **Adsterra**.
2. أضف رابط موقعك (رابط Vercel) في الشبكة الإعلانية للحصول على الموافقة.
3. ستحصل على كود برمجي (Script) للإعلان.
4. افتح ملف `index.html` الموجود في المجلد الرئيسي، وضع كود الـ Script الأساسي للإعلانات داخل علامة `<head>`.
5. افتح ملف `src/App.tsx` وابحث عن المكون المسمى `AdPlaceholder`.
6. استبدل محتوى هذا المكون بالكود الخاص بالوحدة الإعلانية (Ad Unit) الذي أعطتك إياه الشبكة الإعلانية (مثلاً كود `<ins class="adsbygoogle"...>`).
7. قم برفع التعديلات إلى GitHub، وسيقوم Vercel بتحديث الموقع تلقائياً!

## 💻 التشغيل المحلي (للمطورين)

إذا أردت تشغيل وتعديل التطبيق على حاسوبك الشخصي:

1. تأكد من تثبيت Node.js.
2. افتح موجه الأوامر (Terminal) في مجلد المشروع واكتب:
   ```bash
   npm install
   ```
3. قم بإنشاء ملف `.env` في المجلد الرئيسي وأضف فيه مفتاح API:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
4. لتشغيل التطبيق:
   ```bash
   npm run dev
   ```
5. افتح المتصفح على الرابط `http://localhost:3000`.
