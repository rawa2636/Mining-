# GeoVision AI 🌍🔍

**GeoVision AI** هو تطبيق ويب متطور يعتمد على تقنيات الذكاء الاصطناعي (AI) لمساعدة المنقبين والجيولوجيين والمهتمين في تحليل صور الصخور والعينات الجيولوجية بدقة عالية.

## 🌟 مميزات المشروع

* **تحليل ذكي للصور:** رفع صور العينات الصخرية ليقوم الذكاء الاصطناعي بفحصها مجهرياً وبصرياً.
* **اكتشاف المؤشرات الجيولوجية:** التعرف الدقيق على أنواع الأكاسيد (مثل الهيماتيت، الليمونيت، الجويثيت) وعروق المرو (الكوارتز).
* **تقدير احتمالية وجود الذهب:** خوارزميات مدمجة تعطي نسبة مئوية لاحتمالية وجود تمعدن الذهب بناءً على المؤشرات البصرية.
* **خرائط حرارية (Heatmaps):** تحديد أماكن تواجد المؤشرات الهامة على الصورة مباشرة باستخدام مربعات الإحاطة (Bounding Boxes).
* **سجل التحليلات:** حفظ تلقائي لجميع الفحوصات السابقة للرجوع إليها في أي وقت.
* **واجهة مستخدم عربية:** تصميم أنيق وسهل الاستخدام باللغة العربية بالكامل.
